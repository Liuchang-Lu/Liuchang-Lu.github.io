Project Summary

Dec 03, 2019Hits

自我介绍：你好，我叫陆柳畅，来自上海七宝德怀特高级中学，目前高二，在学校学习了电脑编程相关的课程，希望在这个领域有着更深一步的了解。

APP介绍：本app用于学校或是组织的社团招新。不同社团可以在app上发布自己的社团相关信息，学生或成员也可以用其账号在app上进行相关社团的报名。社团信息发布者可以看到报名人数，报名者也可以根据时间来检索相关的社团文章。其中还设有了可以查看所有文章的管理员，方便整个系统的稳定运行。
